# Riri Discord Bot Website

Professional website for the Riri Discord Bot, deployed on GitHub Pages.

## Features
- Modern, responsive design
- Interactive particle background
- Command browser with filtering
- Mobile-friendly layout
- Smooth animations
- Discord-themed color scheme

## Deployment on GitHub Pages

1. **Create a new repository** on GitHub named `riri-website`
2. **Upload all files** to the repository:
   - `index.html` (root)
   - `css/style.css`
   - `js/script.js`
   - `assets/` folder with favicon (optional)

3. **Enable GitHub Pages**:
   - Go to repository Settings
   - Scroll to "Pages" section
   - Select "Deploy from a branch"
   - Choose branch: `main`
   - Select folder: `/ (root)`
   - Click Save

4. **Your site will be live at**: `https://[your-username].github.io/riri-website/`
